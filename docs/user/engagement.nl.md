# Engagement-gids

IntraVox laat je interacteren met pagina's via **reacties** en **comments**. Deze gids legt uit hoe je deze features gebruikt.

**Doelgroep**: alle IntraVox-gebruikers

---

## Overzicht

Engagement-features laten je:

- Reageren op pagina's met emoji's om waardering te tonen
- Comments achterlaten om je gedachten te delen
- Reageren op comments van anderen
- Reageren op comments met emoji's

## Reacties

Reacties zijn een snelle manier om op een pagina te reageren zonder een comment te schrijven.

### Een reactie toevoegen

1. Scroll naar de onderkant van een pagina
2. Vind de reactie-balk met emoji-knoppen
3. Klik op een emoji om je reactie toe te voegen

Beschikbare reacties:

| | | | | | |
|:---:|:---:|:---:|:---:|:---:|:---:|
| 👍 | 👎 | ❤️ | 🎉 | 😊 | 😢 |
| 😮 | 😂 | 🤔 | 👏 | 🙏 | 💪 |
| ✅ | ⭐ | 🔥 | 💯 | 👀 | 🚀 |

### Je reactie wijzigen

Klik op een andere emoji om je reactie te wijzigen. Je vorige reactie wordt automatisch verwijderd.

### Je reactie verwijderen

Klik nogmaals op dezelfde emoji om je reactie volledig te verwijderen.

### Bekijken wie reageerde

Hover over een reactie-aantal om een lijst te zien van mensen die met die emoji hebben gereageerd.

## Comments

Comments laten je gedachten delen en discussies voeren over pagina-inhoud.

### Een comment schrijven

1. Scroll naar de comments-sectie onder de pagina-inhoud
2. Klik op het tekstveld "Schrijf een comment..."
3. Typ je bericht
4. Klik **Verzenden** of druk `Ctrl+Enter` om te posten

### Reageren op een comment

1. Vind de comment waarop je wilt reageren
2. Klik op de **Reageren**-knop onder de comment
3. Een tekstveld verschijnt onder die comment
4. Typ je antwoord
5. Klik **Reageer** of druk `Ctrl+Enter` om te posten

Antwoorden verschijnen ingesprongen onder de originele comment, zodat conversaties makkelijk te volgen zijn.

### Je comment verwijderen

1. Vind je comment
2. Klik op het delete-icoon (prullenbak)
3. Bevestig de verwijdering

Je kunt alleen comments verwijderen die je zelf schreef.

### Comments sorteren

Gebruik het dropdown-menu bovenaan de comments-sectie:

- **Nieuwste eerst**: meest recente comments bovenaan
- **Oudste eerst**: oudste comments bovenaan

## Reacties op comments

Je kunt ook op individuele comments reageren, niet alleen op pagina's.

### Een reactie aan een comment toevoegen

1. Vind de comment waarop je wilt reageren
2. Klik op de emoji-knop (smiley) naast de comment
3. Selecteer je reactie uit de picker

### Een comment-reactie verwijderen

Klik nogmaals op dezelfde emoji om je reactie te verwijderen.

## Tips voor goede engagement

### Wees respectvol

Houd je comments professioneel en constructief. Vergeet niet dat collega's lezen wat je schrijft.

### Blijf on-topic

Reageer op de pagina-inhoud. Niet-gerelateerde vragen? Gebruik andere communicatiekanalen.

### Wees beknopt

Korte, heldere comments werken het best. Kom snel ter zake.

### Gebruik reacties verstandig

Een snelle emoji-reactie toont waardering zonder de comment-telling te verhogen. Gebruik reacties wanneer je het eens bent maar niets nieuws toe te voegen hebt.

### Reageer doordacht

Bij reageren op iemand: zorg dat je waarde toevoegt aan het gesprek.

## Veelgestelde vragen

### Waarom zie ik de reactie-knoppen niet?

Reacties kunnen uit staan voor deze specifieke pagina of voor het hele intranet. Dat is een instelling die door je beheerder of content-managers wordt beheerd.

### Waarom kan ik niet op deze pagina reageren?

Comments kunnen uit staan voor bepaalde pagina's, zoals officiële beleidsdocumenten of aankondigingen. Vraag het je intranet-beheerder.

### Kan ik eigen emoji's gebruiken?

De beschikbare reactie-emoji's worden door IntraVox bepaald. Custom emoji's worden op dit moment niet ondersteund.

### Zijn mijn comments voor iedereen zichtbaar?

Ja, comments zijn zichtbaar voor alle gebruikers met toegang tot de pagina. Let dus op wat je post.

### Kan ik een comment bewerken na posten?

Nee, comments kunnen na posten niet worden bewerkt. Verwijder je comment en post een nieuwe als nodig.

### Hoe meld ik ongepaste comments?

Neem contact op met je intranet-beheerder of IT-afdeling om ongepaste content te melden.

### Wie kan mijn comments verwijderen?

Je kunt je eigen comments verwijderen. Beheerders kunnen ook elke comment verwijderen indien nodig.
